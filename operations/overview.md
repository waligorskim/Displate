## Key Workflows

### Order-to-Delivery
1. Customer places order
2. Payment processed
3. Order enters production queue
4. Item printed/prepared
5. Quality check
6. Shipped to customer
7. Delivery confirmation

[NEEDS INFO: Detailed workflow with timeframes]

### Returns Processing
1. Customer initiates return
2. Return authorized
3. Customer ships item back (at their expense for international)
4. Item received and inspected
5. Refund processed
